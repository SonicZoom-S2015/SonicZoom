Installation and Running:

Only installation required is placing the file in proper folder location.
Requirements: Chrome Browser

For Server use:

1. Download the ZIPed file from GitHub.
2. Unzip the file on server
3. Go to file location using Chrome Browser. The index file will run automatically upon loading into folder location.

For Personal use:
1. Download the ZIPed file from GitHub.
2. Unzip the file into desired location on computer.
3. Double-click the index.html file
4. Make sure the default browser is Chrome. If it is not, copy the link in the URL of your default browser upon opening the index.html file. Paste it into your Chrome Browser. Press enter and enjoy!
